# Rodando o projeto

#### Primeiro clone o projeto:

```bash
$ git clone https://github.com/MRLSK8/Pokedex.git 
```

#### Rode o comando para instalar os pacotes:

```bash
$ npm install
```

#### Rode o projeto:

```bash
$ expo start
```
